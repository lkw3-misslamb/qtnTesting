Miss Lam's Testing and Sharing Space

示範用 Prompt：

請協助我撰寫一個簡單的 HTML 網站首頁（index.html）。

需求如下：
- 網頁標題（title）為：「Miss Lam's Testing and Sharing Space」
- 頁面中需要有兩個超連結：
  1. 連結到 page1.html
  2. 連結到 page2.html
- 請加入一個頁面主標題（heading）
- 最後只輸出 HTML 原始碼
    
Create web app，用 svg 繪製一個可交互動態示範：
拖動三個頂點來改變三角形的形狀，並在畫面上即時計算與更新三個角度的數值。
請以 HTML + CSS + JavaScript 完成，使用 SVG 繪製三角形與頂點控制點，
拖動頂點時三角形需立即重繪並更新角度標示。界面需簡潔，角度以文字標示在各頂點旁。
